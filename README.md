# 🚀 PercentagePro - Premium Online Calculator Website

PercentagePro is a modern, fast, and responsive online calculator website built using **HTML, CSS, and JavaScript**. It provides multiple calculators in one place with a clean UI and mobile-friendly design.

🌐 **Live Demo:** https://karan.github.io/Age-Calculator-Pro/

---

## ✨ Features

- 📊 Percentage Calculator
- 💰 GST Calculator
- 🏷️ Discount Calculator
- 🎂 Age Calculator
- ⚖️ BMI Calculator
- 🎓 CGPA Calculator
- 💳 EMI Calculator
- 🏦 Loan Calculator
- 🧮 Scientific Calculator

### Additional Features

- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🔍 Calculator Search
- 🗂️ Category Filter
- 📋 Copy Result
- 📤 Share Result
- 💾 Calculation History
- ⚡ Fast Performance
- 🎨 Modern Premium UI

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/karan/Age-Calculator-Pro.git
```

Open the project folder in **Visual Studio Code**.

### 3. Run

Use **Live Server** or open `index.html` in your browser.
